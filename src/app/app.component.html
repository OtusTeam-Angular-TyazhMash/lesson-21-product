RxJS Overview

Продуктовая задача

Я как пользователь хочу искать продукты вводя символы в строку поиска. Я хочу обновить результат поиска продуктов, нажав на кнопку Refresh.

Критерии качества
  1. Оптравлять запрос на сервер через 500мс, как пользователь закончил ввод названия продукта в элемент поимка
  2. Если пользователь ввёл строку поиска, после удалил и сразу ввёл туже строку, то запрос на север не отправляется
  3. Пользователь кликну на кнопку Refresh, данные продуктов обновились в соотвествии с последним значением из элемента поиска
  4. Если поле поиска, то нажатие на кнопку Refresh не выполняет запрос на сервер
